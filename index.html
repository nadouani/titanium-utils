Welcome to Titanium Utils project page.

Titanium Utils is a Javascript library build using Grunt.

Titanium Utils is designed to provide easy way to play with Javascript data types:

  • Objects
  • Arrays
  • Strings
  • Dates
  • Numbers
  • ... and Functions

Titanium Utils aims also to provide some data validation utilities:

  • Date Formats
  • URLs, Email

Project build information

Titanium Utils is build using Travis CI
